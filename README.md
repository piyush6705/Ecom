# E-Commerce Project

A modern e-commerce web application built with **React**, **Vite**, and **React Router**. Users can browse products, add items to the cart, update quantities, manage their cart, and choose delivery options during checkout.

---

## Overview

This project demonstrates the fundamentals of building a client-side e-commerce application using React. It focuses on reusable components, routing, state management, API integration, and testing.

---

## Features

- Browse products
- View product information
- Add products to cart
- Remove products from cart
- Update product quantity
- Select delivery options
- Dynamic order summary
- Responsive user interface
- API integration using Axios

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ecommerce-project.git
```

Navigate to the project folder:

```bash
cd ecommerce-project
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```



## Running Tests

```bash
npm test
```

or

```bash
npm run test
```
